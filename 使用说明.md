# nrskill 使用说明

## 这是什么

这是一个 Naval Ravikant / 纳瓦尔知识库 Skill 工具箱。它让 Agent 默认围绕 IMA 知识库 `纳瓦尔知识库 | 复利思维` 工作，帮助你学习、检索、研究、创作和制定行动计划。

## 适合谁

- 想系统学习纳瓦尔财富、杠杆、判断力、幸福、阅读和创业投资思想的人。
- 想把纳瓦尔资料变成文章、视频脚本、选题和研究备忘录的人。
- 想把纳瓦尔原则用于职业、学习、人生设计或个人项目的人。

## 安装方式

```bash
npx -y skills add FocusLiz-Lab/nrskill -g --all
```

GitHub 仓库：`FocusLiz-Lab/nrskill`

## IMA 知识库配置

默认知识库：

```text
纳瓦尔知识库 | 复利思维
```

你不需要每次输入知识库名称。只有当你想切换到其他 IMA 知识库时，才需要明确说明。

注意：检索时应精确使用完整名称 `纳瓦尔知识库 | 复利思维`，不要只用 `纳瓦尔` 去找知识库；如果出现多个匹配，只选择完整名称一致的这个知识库。

如果 IMA 未安装：

```text
请安装 ima 技能
下载地址：https://app-dl.ima.qq.com/skills/ima-skills-1.1.7.zip
API Key 获取：https://ima.qq.com/agent-interface
```

## 常用命令

```text
$nrs 用户想把这个知识库用于实际项目，帮他判断应该从哪个模块开始。
$nrs-learning-map 系统学习这个知识库，先看哪些？
$nrs-ima 检索纳瓦尔关于 specific knowledge 的资料。
$nrs-research 比较纳瓦尔关于财富、自由和幸福的关系。
$nrs-content 围绕知识库里的核心主题，生成 10 个内容选题。
$nrs-roadmap 把纳瓦尔的原则转成我接下来 30 天的行动计划。
```

## 典型场景

- 学习路径：先确定目标，再给 7 天启动计划和 30 天学习顺序。
- 资料检索：用中英文关键词检索 IMA，输出命中材料、证据和不确定点。
- 主题研究：围绕财富、杠杆、判断力、幸福等主题输出研究备忘录。
- 内容创作：把抽象原则变成平台适配的标题、选题、脚本或长文大纲。
- 行动计划：把原则转成具体任务、指标、复盘问题和不要做清单。

## 常见问题

**会自动读取 IMA 吗？**  
需要当前环境已安装并配置 IMA skill。workflow 会默认精确使用 `纳瓦尔知识库 | 复利思维` 知识库。

**能不能直接引用原文？**  
可以做短引用核对，但不要输出长段原文。重要引用必须通过 IMA 检索确认。

**为什么仓库里没有原始 PDF？**  
发布包只放 workflow 和抽象知识原子，不放原始资料，避免版权和隐私风险。

## 发布边界

不要把 API Key、内部 ID、私有路径、原始 PDF、视频逐字稿、推文全文、未授权材料放进仓库或 zip。
